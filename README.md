# Spotify Sentiment Analysis

Proyek Data Mining untuk menganalisis sentimen ulasan pengguna Spotify menggunakan teknik text mining dan klasifikasi machine learning.

## Tools & Libraries
- Python 3.11
- Pandas, NumPy
- NLTK
- Scikit-learn
- Google Play Scraper

## Cara Menjalankan
1. Clone repository
2. Buat virtual environment
3. Install dependencies:
   pip install -r requirements.txt
4. Jalankan notebook / script
